# Awesome Preview

Awesome Preview 是一个浏览器扩展，旨在提供更好的浏览器的预览体验。

![Awesome Preview Logo](path/to/logo.png)

## 官网
[awesome-preiew](https://awesome-preiew.pages.dev/)

## 功能特点

- 智能模式：自动调整预览位置
- 固定模式：在指定位置显示预览
- 居中模式：在屏幕中央显示预览
- 黑名单功能：可以设置不需要预览的网站
